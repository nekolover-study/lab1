Лабораторная работа №1 Данный проект представляет собой реализацию функций для вычисления геометрических параметров различных фигур на языке C++ Работа выполнена в рамках учебного курса по программированию.

Цель работы: изучение платформы GitHub и практическое использование Git для хранения, обновления и распространения исходного кода проекта. реализация алгоритмов вычисления площади, периметра и других характеристик геометрических фигур по заданным входным данным.

Функционал: В рамках данной лабораторной работы были выбраны и реализованы следующие фигуры:

Прямоугольник Периметр: P=2*(a+b) Площадь: S=a*b Длина диагонали: d=(a^2+b^2)^(1/2)​

Треугольник Периметр: P=a+b+c Площадь (Формула Герона): S=(p(p−a)(p−b)(p−c))^(1/2)​, где p — полупериметр. Проверка на равнобедренность: Определение, являются ли хотя бы две стороны треугольника равными.​
